⚙️ Things to Be Trained In / Practice With

Hands-on tools and skills where doing matters more than reading.

Technical Tools & Skills

  • Python (basic) — run scripts, use pandas for small data tasks.
  • Microsoft Power Automate / Power Apps — build flows and simple app connectors.
  • ChatGPT / Copilot / Azure AI Studio — experiment with prompts and small integrations.
  • Data visualization — Excel, Power BI, or Google Data Studio for telling a story with data.
  • Version control — basic GitHub actions: clone, commit, push, pull.
  • Documentation writing — create clear step-by-step guides and READMEs.

Practical Ideas

  • Use pandas to clean a CSV and plot a chart with matplotlib.
  • Create a Power Automate flow that emails you when a new file is added to OneDrive.
  • Write a prompt that converts a long policy into a short summary and test it in ChatGPT.

Where to Practice

Try small projects on Kaggle, the free tiers of cloud platforms, Microsoft Learn, and GitHub repositories for version control practice.
