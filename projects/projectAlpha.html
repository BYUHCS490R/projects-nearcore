Internet

This is the big worldwide network that connects computers

Web

The websites and pages we visit using the Internet

How are the Internet and Web different?

Internet is similar to roads, while web is similar to the houses on the roads

ASCII

American Standard Code for Information Interchange, an old system for English letters and numbers.

Unicode

New system that works for all languages and emojis.

What is UTF-8

Unicode Transformation Format - 8-bit. The common way to store Unicode text.

URL

Uniform Resource Locator. The web address of a page.

Domain Name

The easy name (like google.com) instead of a number.

HTTP

HyperText Transfer Protocol. The rules for loading web pages.

HTTPS

Hypertext Transfer Protocol Secure. Same as HTTP but safe and encrypted.

IP Address

Internet Protocol Address. The number ID for a computer or website.

DNS

Domain Name System. The Internet’s phonebook — turns names into numbers.

W3C Recommendation

World Wide Web Consortium. The rules that keep the Web working the same everywhere.

XML

Extensible Markup Language. Used for storing and sharing data.

XHTML

Extensible HyperText Markup Language. Stricter HTML with XML rules.

HTML

HyperText Markup Language. The basic code for the web pages.

HTML5

The newest HTML, with video, audio, and modern features. The number "5" indicates that it is the fifth major version of the HTML standard.
