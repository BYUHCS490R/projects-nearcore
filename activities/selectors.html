Q&A

  1. List out the different ways to select an item.
    • Tag name
    • ID
    • Class
    • Attribute
    • Pseudo-classes
  2. What is the difference between an ID and a Class?

    3. ID is unique for one element, while class can be used on many elements.

  3. What is the difference between a SPAN tag and a DIV tag?

    4. SPAN is inline for small text, while DIV is block for larger sections.

  4. What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

    5. DIV is a generic container, while HEADER, MAIN, and FOOTER show content meaning.

  5. What are the different ways to select a color?
    • Name
    • Hex
    • RGB
    • RGBA
    • HSL
