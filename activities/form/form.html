The form tag has an "action" attribute. What does it do?

The action attribute specifies the URL where the form's collected data will be sent for processing when the form is submitted.

The form tag has a "method" attribute. What does it do?

The method attribute specifies the HTTP method (most commonly GET or POST) that the browser will use to send the form data to the server.

What is the difference between a "POST" and a "GET"?

GET is used to retrieve data, appending form values to the URL, which makes them visible and limits the data size.

POST is used to submit or create/update data, sending form values in the request body, which is more secure for sensitive or large amounts of data.
