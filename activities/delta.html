What are special entity characters?

These are codes that let you display symbols or reserved characters (like <, >, or ©) in an HTML page that would otherwise be misinterpreted by the browser.

What are phrase tags?

Phrase tags are used to give emphasis on a specific text style or meaning.

What is the difference between OL and UL Tags?

UL (Unordered List) tags organize the data in bulleted list. Example of this is below:

  • Castles
  • Genagators
  • Turtle Bay
  • Sunset
Meanwhile OL (Ordered List) tags organize the data by using number list, alphabetical order list or roman numerical list. Example of this is below:
  1. Castles
  2. Genagators
  3. Turtle Bay
  4. Sunset
