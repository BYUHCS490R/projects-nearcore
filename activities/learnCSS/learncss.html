Beginning CSS

CSS means Cascading Style Sheets. It is used to style and layout web pagesâ€”for example, to alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features.

The difference between inline, embedded and external CSS.

  • Inline CSS: Inline CSS is used to apply a unique style to a single HTML element. To use inline styles, add the style attribute to the relevant element.
  • Embedded CSS: Embedded CSS is used to define a style for a single HTML page.
  • External CSS: External CSS is used to define the style for many HTML pages. With an external style sheet, you can change the look of an entire website by changing just one file.

Why External CSS should be used?

  • External CSS is more efficient because it can be cached by the browser, reducing load times for subsequent pages.
  • It allows for better organization and separation of content (HTML) and presentation (CSS).
  • External CSS makes it easier to maintain and update styles across multiple pages.

What is a selector used for?

  • A selector is used to "select" the HTML elements you want to style. It can target elements by their name, class, id, attribute, and more.
