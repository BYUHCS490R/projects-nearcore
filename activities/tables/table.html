Q&A

When do you use a table?

Use a table when you need to organize structured data that has a meaningful relationship, usually in two dimensions (rows and columns).

When do you not use a table?

Don't use a table for simple lists, paragraphs of text, or just for page layout (like trying to put an image next to text).

When do you need to do extra work for table accessibility?

Every time a table is used, it means using special HTML tags (like for headers) so screen readers can properly announce what each number or word in the grid means. For tables with complex, nested headers, you have to do even more work to link the cells correctly.
