Q & A

What is the root?

The root is the project folder, where the website starts.

What is the relationship between folder and file names and the path?

The path is the location of a file or folder in relation to the root directory.

What is a default document?

It is the file that's named index.html.

Explain the difference between an absolute link and a relative link.

  • Absolute Link Example: https://www.example.com
  • Relative Link Example: page.html

An absolute link provides the full, unique web address (URL), while a relative link provides only the path to a resource from its current location.

Why would you use relative links?

Because relative links make it easier to manage and update the links when the site structure changes.

List the 3 rules with relative links.

  • Link to a file in the same folder
  • Link to a file in a subfolder
  • Link to a file in a parent folder
